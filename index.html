About

Nama saya Achmad Shiddiqi Aulia, saya sangat tertarik dengan dengan teknologi informasi sejak kelas 5 sd. Sejak saat itu ayah saya membelikan komputer yang saya gunakan untuk belajar dan juga bermain. Ketertarikan saya terhadap tekonologi menumbuhkan keinginan saya untuk berkuliah di jurusan sistem informasi atau teknik informatika, karena itu pada masa SMA saya belajar semaksimal mungkin agar diterima di universitas negeri. Alhamdulillah saya pun berkuliah di UIN Syarif Hidayatullah Jakarta di bidang studi sistem informasi.

Saat ini saya beranjak ke semester 8. Saya mulai serius untuk mendalami ilmu Web Development saat semester 5. Beberapa bahasa pemrograman web saya pelajari seperti HTML, CSS, PHP, Bootstrap, dan Javascript dasar. Selain itu saya juga sedang belajar menggunakan framework Code Igniter dan juga Laravel. Namun ilmu saya pada PHP dan Javascript masih sangat dasar, saya ingin belajar lebih lanjut pada bagian tersebut. Saya juga pernah mengerjakan beberapa tugas kelompok sebagai web developer pada beberapa mata kuliah.
